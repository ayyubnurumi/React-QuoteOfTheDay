## 👨🏻‍💻 Run Locally

- Clone the project

```bash
  git clone https://github.com/Th3Wall/Fakeflix
```

- Go to the project directory

```bash
  cd fakeflix
```

- Install dependencies

```bash
  npm install
```

- Create a .env file

- Request an API key from RapidAPI and then add it to the .env file

```
REACT_APP_API_KEY= "your-React-App-API-key"
```

- Start the server

```bash
  npm start
```
